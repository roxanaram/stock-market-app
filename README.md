# StockScope Final Project

StockScope contains the responsive web app and the Flutter mobile app for the Frontend Programming final project.

## Structure

- Root folder: responsive web app published with GitHub Pages.
- `mobile-app/`: Flutter native app for Android/iOS emulator or device.
- `documentation/`: project documentation and screenshots can be added here.

## Main features

- Stock market search by company, ticker, country, and market.
- Device/location based country selection on web and mobile.
- Market snapshot with stock prices and news.
- Detailed stock page with tables, ChartJS/fl_chart charts, time ranges, and news links.
- Currency switching.
- Portfolio/watchlist page.
- Pro subscription page.
- Shared dark visual style across web and mobile.

## GitHub Pages

Publish this repository from the root folder using GitHub Pages. If the repository name is not `stock-market-final`, update the website URL inside the Flutter files before the final demo.
